# Token List for TTwof's DEX
Data for TTwof

Template : 
```json
{
	"name": "<name>",
	"symbol": "<symbol>",
	"tokenAddress": "<0x...>",
	"website": "https://example.com",
	"blockExplorer": "https://viewblock.io/thundercore/address/<0x...>"
	"icon": "<Logo url>.<png/svg>"
}
```

Example : 
```json
{
	"name": "TT",
	"symbol": "TT",
	"tokenAddress": "TT",
	"website": "https://thundercore.com",
	"blockExplorer": "https://viewblock.io/thundercore/"
	"icon": "https://wtt.thundercore.com/favicon.ico"
}
```
